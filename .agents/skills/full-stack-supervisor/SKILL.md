# Full-Stack Supervisor

Master orchestration skill that coordinates multiple sub-skills to fulfill complex requests efficiently.

## Orchestration Logic
- **Frontend requests**: Chain `frontend-design` + `ui-ux-pro-max` + `accessibility-audit`.
- **Backend requests**: Chain `postgres-patterns` + `api-design` + `security-scanner`.
- **Deployment requests**: Chain `docker-compose` + `ci-pipeline` + `monitoring-setup`.

## Workflow Control
1. Assess requested task.
2. Select target bundle/skills.
3. Execute sequential pipeline phases.
4. Run validation checks.
