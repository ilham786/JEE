# Startup Founder Mode

Guides the creation of minimal viable products, high-converting SaaS landing pages, onboarding loops, and analytics integrations.

## Core Pillars
1. **MVP Simplicity**: Build core value propositions fast.
2. **Landing Page Mastery**: High-impact messaging, dark glassmorphism, clear CTA.
3. **Metrics-Driven**: Clean analytics hooks.
